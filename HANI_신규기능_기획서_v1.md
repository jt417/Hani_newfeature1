## 1. 기능 개요

### 1-1. 배경 및 목적

사주명리학에서 대운·세운·월운·일운은 형(刑)·충(沖)·합(合)·파(破)·해(害) 등의 상호작용을 통해 크고 작은 이벤트를 만들어낸다. 그러나 사용자 대부분은 이 상호작용이 자신의 삶에 *구체적으로 어떻게* 작용하는지 알지 못한다.

**사주일기**는 일상의 사건을 사주 상호작용과 매핑하여 기록함으로써, 사용자가 자신의 사주 패턴을 스스로 학습하고, 미래 예측의 정확도를 높이는 데 기여한다.

**다자간 궁합**은 20~30대가 친구, 연인, 동료와 함께 사주를 즐길 수 있는 소셜 기능으로, 앱의 바이럴 확산과 유저 확보를 목표로 한다.

### 1-2. 타겟 사용자

| 기능 | 타겟 | 사주 지식 수준 |
|------|------|---------------|
| 사주일기 | 자기 이해·자기 관리에 관심 있는 사용자 | 초보 (십성/합충 개념 모름) |
| 다자간 궁합 | 20~30대, 모임·파티·소개팅 문화 | 초보~중급 |

### 1-3. 앱 내 진입점

하단 탭바에 **"사주일기"** 탭과 **"궁합"** 탭을 신규 추가한다.

## 2. 사주일기 (사주 다이어리)

### 2-1. 핵심 컨셉

> "1분 기록 → 나의 사주 패턴 발견 → AI가 더 정확한 분석 제공"

사용자가 매일 짧은 기록을 남기면, 앱이 해당 시점의 사주 상호작용(형충합파해)과 자동으로 매핑한다. 축적된 데이터는 AI 챗봇(HANI PASS 연동)의 분석 정확도를 높이는 시드 데이터가 된다.

### 2-2. 두 가지 기록 모드

#### A. 일일 다이어리 (Daily Diary)

일운 단위의 미세한 변화를 포착하는 일기. 매일 1분 남짓으로 빠르게 기록한다.

**입력 항목:**

| 항목 | 형식 | 설명 |
|------|------|------|
| 날짜 | 날짜 선택 | 기본값: 오늘. 과거 날짜도 선택 가능 |
| 감정 상태 | 0~5 스케일 (이모지/슬라이더) | 0 = 최악, 5 = 최고 |
| 체감 강도 | 0~5 스케일 | 오늘 사건의 영향력을 체감한 정도 |
| 사건 카테고리 | 다중 선택 (아래 표 참조) | 복수 선택 가능 |
| 메모 | 자유 텍스트 (300자 제한) | 어떤 일이 있었는지 간단 메모 |

**사건 카테고리 (십성 기반 — 초보 친화 레이블):**

| 레이블 (사용자 노출) | 십성 | 설명 / 예시 |
|----------------------|------|------------|
| 직장생활 (정관) | 正官 | 업무, 승진, 조직 내 역할 변화 |
| 건강·돌발상황 (편관) | 偏官 | 사고, 갑작스러운 건강 이슈, 예상 밖 상황 |
| 돈 관리 (정재) | 正財 | 월급, 저축, 안정적 수입 관련 |
| 투자·사업 (편재) | 偏財 | 주식, 부업, 예상 밖 수입/지출 |
| 공부·시험 (정인) | 正印 | 자격증, 학습, 멘토 관련 |
| 고민·방황 (편인) | 偏印 | 진로 고민, 우울, 불안, 생각 과잉 |
| 취미·여행 (식신) | 食神 | 맛집, 여행, 여가, 즐거운 경험 |
| 다툼·이별 (상관) | 傷官 | 갈등, 말실수, 관계 변화 |
| 경쟁·자존심 (비견) | 比肩 | 경쟁상황, 자기주장, 동료 관계 |
| 손해·후회 (겁재) | 劫財 | 금전 손실, 사기, 후회되는 선택 |

> **UX 포인트:** 십성 용어는 괄호 안에만 표시하고, 사용자가 탭하면 "정관이란?" 같은 짧은 툴팁을 보여준다. 초보도 자연스럽게 십성 개념을 학습할 수 있다.

**사주 정보 자동 표시 (일운):**

일일 다이어리 화면 상단에 오늘의 사주 상호작용을 자동으로 표시한다.

```
┌─────────────────────────────────────┐
│  📅 2026년 3월 28일 (토)             │
│                                     │
│  오늘의 일운: 丙午 (병오)             │
│  ─────────────────────────          │
│  🔄 내 사주와의 상호작용:             │
│  • 午-子 충(沖) → 일지 충격           │
│  • 丙-辛 합(合) → 일간과 합           │
│                                     │
│  💡 "오늘은 감정 변화가 클 수 있어요"  │
│     (탭하여 자세히 보기)              │
└─────────────────────────────────────┘
```

- 상호작용의 의미를 **초보 친화적 한 줄 요약**으로 제공
- 탭하면 해당 형충합파해의 상세 설명 팝업
- 사주 정보는 앱이 자동 계산하여 표시 (사용자 입력 불필요)

#### B. 과거 사건 기록 (Life Event Log)

대운·세운·월운 단위의 큰 사건을 기록하는 모드. 인생의 주요 전환점을 기록하여 사주 패턴을 역추적한다.

**입력 항목:**

| 항목 | 형식 | 설명 |
|------|------|------|
| 시기 | 년/월 선택 | 일 단위는 불필요 (기억 부정확) |
| 사건 유형 | 다중 선택 (일일 다이어리와 동일 카테고리) | |
| 사건 제목 | 텍스트 (50자) | 예: "첫 이직", "큰 수술" |
| 상세 내용 | 자유 텍스트 (500자) | 구체적인 상황 기술 |
| 감정 회고 | 긍정/부정/중립 | 당시의 전반적 감정 |
| 영향도 | 1~5 스케일 | 인생에 미친 영향력 |

**사주 상호작용 자동 매핑:**

사용자가 년/월을 입력하면, 앱이 해당 시기의 대운·세운·월운을 자동 계산하여 형충합파해를 표시한다.

```
┌─────────────────────────────────────┐
│  📌 2023년 7월                       │
│                                     │
│  대운: 壬申 (임신) — 2019~2028       │
│  세운: 癸卯 (계묘) — 2023년          │
│  월운: 己未 (기미) — 7월             │
│                                     │
│  🔍 감지된 상호작용:                  │
│  ┌───────────────────────────┐      │
│  │ ★★★ 대운 申-寅 충(沖)     │ ← 영향력 大  │
│  │ ★★  세운 卯-酉 충(沖)     │ ← 영향력 中  │
│  │ ★   월운 未-丑 충(沖)     │ ← 영향력 小  │
│  └───────────────────────────┘      │
│                                     │
│  ✏️ [수정하기] — 잘못된 매핑 수정 가능 │
└─────────────────────────────────────┘
```

> **운의 영향력 위계:** 대운 >>>> 세운 >> 월운 > 일운
> 동일한 "진술충"이라도 대운에서 발생하면 인생 전환급 이벤트이고, 일운에서 발생하면 하루의 작은 불편함 수준이다. UI에서 별 개수(★)나 색상 강도로 영향력 차이를 시각적으로 구분한다.

**사용자 확인/수정 프로세스:**
1. 앱이 자동 계산한 상호작용 목록을 표시
2. 사용자가 "이건 아니다" 싶으면 삭제하거나, 빠진 상호작용을 추가 가능
3. 수정 이력도 저장 → AI 학습 데이터로 활용

### 2-3. 기록 유도 시스템

**① 푸시 알림**
- 매일 저녁 (기본 21:00, 사용자 설정 가능) "오늘 하루 어땠나요?" 푸시
- 특별한 상호작용이 있는 날: "오늘 午-子 충이 있어요. 어떤 일이 있었나요?" 맞춤 알림

**② 앱 접속 시 팝업**
- 어제 기록을 안 했으면 소프트 팝업: "어제는 어떤 하루였나요? (30초면 충분해요)"
- 강제성 없이 "나중에" 버튼 제공

**③ 스트릭(연속 기록) 게이미피케이션**

| 스트릭 | 보상 |
|--------|------|
| 7일 연속 | 🔥 7일 배지 + AI 간단 분석 1회 무료 |
| 30일 연속 | 🌟 30일 배지 + 월간 리포트 자동 생성 |
| 100일 연속 | 💎 100일 배지 + HANI 재화 보너스 |
| 365일 연속 | 🏆 연간 마스터 배지 + 프리미엄 분석 1회 |

- 스트릭 카운터를 사주일기 메인 화면 상단에 눈에 띄게 표시
- 스트릭 깨지기 전 알림: "🔥 연속 23일! 오늘도 기록해서 이어가세요"

**④ 운 변화 알림**

| 운 종류 | 알림 타이밍 | 알림 내용 |
|---------|-----------|----------|
| 대운 변화 | 변화 1개월 전 + 당일 | "대운이 바뀌어요! 지난 대운의 기록을 돌아보세요" |
| 세운 변화 | 입춘 전후 | "새해 운이 시작됩니다. 올해의 주요 상호작용을 확인하세요" |
| 월운 변화 | 절기 기준 매월 | "이번 달 운이 바뀌었어요. 어떤 변화가 있을지 체크!" |

### 2-4. 데이터 시각화 및 분석

#### A. 감정 그래프 (주간/월간)
- X축: 날짜, Y축: 감정 스코어(0~5)
- 해당 날짜에 발생한 형충합파해를 그래프 위에 아이콘으로 표시
- 감정 저점에 어떤 상호작용이 있었는지 패턴 파악 가능

#### B. 십성별 이벤트 분포
- 도넛 차트 또는 바 차트
- 카테고리별 이벤트 빈도와 해당 시기의 사주 상호작용 매핑
- 예: "정관(직장생활) 이벤트 12건 중 8건이 관성 충(沖) 시기에 발생"

#### C. 캘린더 히트맵
- 월간 캘린더 뷰
- 각 날짜를 감정 스코어에 따라 색상 그라데이션으로 표시
- 형충합파해가 있는 날짜에 마커 표시
- 한눈에 "이 달은 언제 힘들었는지" 파악 가능

#### D. AI 인사이트 (HANI PASS 연동)
- 축적된 데이터를 기반으로 AI가 패턴 분석
- 예: "당신은 午-子 충이 오면 직장에서 갈등이 생기는 경향이 있어요"
- 예: "과거 3번의 寅-申 충 시기에 모두 이직을 했네요. 다음 寅-申 충은 2028년입니다"
- HANI PASS 구독 시 AI 챗봇이 사주일기 데이터를 참조하여 더 정밀한 상담 제공

```
[하단 탭: 사주일기]
    │
    ├── 메인 화면
    │   ├── 스트릭 카운터 (상단)
    │   ├── 오늘의 일운 + 상호작용 요약
    │   ├── [오늘 기록하기] CTA 버튼
    │   └── 최근 기록 타임라인 (스크롤)
    │
    ├── 일일 기록 화면

    │   ├── Step 2: 체감 강도 (슬라이더)
    │   ├── Step 3: 카테고리 선택 (칩/태그)
    │   ├── Step 4: 메모 (선택)
    │   └── [저장] → 오늘의 상호작용 자동 매핑 확인 화면
    │
    ├── 과거 사건 기록 화면
    │   ├── 년/월 선택
    │   ├── 사건 정보 입력
    │   ├── 자동 매핑된 대운/세운/월운 상호작용 표시
    │   ├── [수정하기] → 상호작용 추가/삭제
    │   └── [저장]
    │
    └── 분석 대시보드
        ├── 감정 그래프 (주간/월간 전환)
        ├── 십성별 이벤트 분포
        ├── 캘린더 히트맵
        └── AI 인사이트 (HANI PASS 전용)
```

## 3. 다자간 궁합

### 3-1. 핵심 컨셉

> "친구들과 함께 보는 사주 — 우리 모임의 케미를 알아보자"

20~30대가 오프라인 모임이나 온라인에서 함께 사주 궁합을 확인하고, 서로의 케미와 역할을 재미있게 알아보는 소셜 기능.

### 3-2. 궁합 유형 구분

| 유형 | 인원 | 진입점 | 비고 |
|------|------|--------|------|
| 1:1 궁합 | 2인 | 궁합 탭 > "1:1 궁합" | 별도 화면, 초대 불필요 (기존 기능 확장 또는 신규) |
| 다자간 궁합 | 3~5인 | 궁합 탭 > "그룹 궁합" | 방 생성 → 초대 → 결제 → 결과 |

> 다자간 궁합은 우선 3~5인으로 기획하며, 추후 인원 확대를 고려한 확장 가능 구조로 설계한다.

### 3-3. 모임 카테고리

| 카테고리 | 아이콘 | 결과 톤앤매너 |
|---------|--------|-------------|
| 친구 | 🤝 | 캐주얼, 재미 위주 |
| 연인 | 💕 | 로맨틱, 관계 심화 |
| 직장 동료 | 💼 | 프로페셔널, 협업 관점 |
| 가족 | 🏠 | 따뜻한, 가족 역학 관점 |
| 사업 파트너 | 🚀 | 비즈니스, 시너지 관점 |
| 기타 (직접 입력) | ✏️ | 범용 |

> 카테고리에 따라 결과 화면의 분석 관점과 어조가 달라진다.

### 3-4. 유저 플로우 — 방 생성부터 결과까지

```
[방장]                              [참여자]
  │                                    │
  ├─ 궁합 탭 > "그룹 궁합"             │
  ├─ 모임 카테고리 선택                 │
  ├─ 방 이름 입력                      │
  │   (예: "대학 동기 모임")            │
  ├─ 인원 설정 (3~5인)                 │
  ├─ [방 만들기]                       │
  │                                    │
  ├─ 대기실 화면                       │
  │   ├─ 초대코드 표시 (6자리, 30분 만료)│

  │   ├─ 링크 공유 버튼                 │

  │                                    │
  ├─ 결제 화면                         │
  │   ├─ 결제 방식 선택                 │

  │   ├─ HANI 재화 사용                 │

      ├─ [공유 결과] 모두 같은 화면      │
```

### 3-5. 대기실 화면 상세

```
┌─────────────────────────────────────┐
│  🎯 대학 동기 모임 (친구)             │
│  ─────────────────────────          │
│  인원: 2/4명 입장 완료                │
│                                     │
│  👤 김민수 (방장) ✅                  │
│  👤 이지은 ✅                        │
│  👤 대기 중...                       │
│  👤 대기 중...                       │
│                                     │
│  ┌─────────────────────────┐        │
│  │   초대코드: A3K-9F2     │        │
│  │   ⏰ 27분 32초 남음     │        │
│  └─────────────────────────┘        │
│                                     │
│  [QR코드 보기]  [링크 복사]           │
│                                     │
│  ──── 방장 전용 ────                 │
│  [인원 변경]  [참여자 내보내기]        │
│  [방 삭제]                           │
└─────────────────────────────────────┘
```

**초대코드 규칙:**
- 6자리 영숫자 (대문자+숫자, 혼동 문자 제외: O/0, I/1/L)
- 30분 후 자동 만료 → 갱신 버튼으로 재생성 가능
- QR코드: 오프라인 모임 시 바로 스캔 → 앱 딥링크

**미가입자 처리:**
1. 초대코드/QR/링크 접속
2. 앱 미설치 → 앱스토어/플레이스토어 이동
3. 미가입 → 회원가입 화면 (간편가입 지원)
4. 가입 완료 후 → 사주 정보 입력 → 방 자동 입장

### 3-6. 결제 시스템

**가격 정책:** 인당 고정 가격 (HANI 재화 기준)

| 결제 방식 | 설명 |
|----------|------|
| 방장 전체 결제 | 방장이 (인당 가격 × 인원수) 일괄 결제 |
| 더치페이 | 각 참여자가 본인 몫을 개별 결제 |
| 지정 결제 | 특정 참여자가 다른 사람의 몫까지 대신 결제 |

**결제 플로우:**
1. 모든 인원 입장 완료
2. 방장이 결제 방식 선택
3. 더치페이/지정결제 시: 해당 참여자에게 결제 요청 푸시
4. 모든 결제 완료 → 결과 생성 시작

**런칭 이벤트 (선택):**
- 다자간 궁합 할인 이벤트 (예: 첫 달 50% 할인)
- 추천인 코드로 방 생성 시 추가 할인

### 3-7. 결과 화면 구성

결과는 **공유 결과**와 **개인 결과** 두 영역으로 나뉜다.

#### A. 공유 결과 (모두 동일하게 보는 화면)

모든 참여자가 동일한 내용을 확인한다. 함께 보며 이야기할 수 있는 콘텐츠.

**포함 요소:**

| 요소 | 설명 |
|------|------|
| 그룹 케미 점수 | 모임 전체의 조화도 (100점 만점) |
| 관계 매트릭스 | N×N 매트릭스 형태로 각 쌍의 궁합 점수 표시 |
| 그룹 역할 배정 | 각자의 모임 내 역할 (예: 분위기 메이커, 중재자, 리더, 아이디어 뱅크 등) |
| 베스트 케미 조합 | "이 그룹에서 최고의 케미" 하이라이트 |
| 그룹 조언 | 카테고리별 맞춤 조언 (친구 모임 vs 직장 vs 연인 등) |

**관계 매트릭스 예시 (4인):**

```
         민수    지은    현우    수빈
민수      —      87     72     91
지은     87       —     95     68
현우     72      95      —     84
수빈     91      68     84      —

🏆 베스트 케미: 지은 & 현우 (95점)
💡 주의 조합: 지은 & 수빈 (68점) — 서로 배려하면 좋은 관계!
```

**그룹 역할 카드:**

```
┌────────────────┐  ┌────────────────┐
│  👑 민수        │  │  🎨 지은        │
│  "든든한 리더"   │  │  "아이디어뱅크"  │
│  이 모임의 기둥  │  │  새로운 걸      │
│  역할이에요     │  │  제안해요       │
└────────────────┘  └────────────────┘
┌────────────────┐  ┌────────────────┐
│  🤝 현우        │  │  🎉 수빈        │
│  "평화의 중재자" │  │  "분위기메이커"  │
│  갈등을 풀어주는 │  │  웃음을 만드는   │
│  존재예요       │  │  존재예요       │
└────────────────┘  └────────────────┘
```

#### B. 개인 결과 (각자만 보는 화면)

각 참여자에게만 보이는 개인화된 분석과 조언.

**포함 요소:**

| 요소 | 설명 |
|------|------|
| 나와 각 멤버의 상세 궁합 | 1:1 궁합의 요약 버전 (클릭 시 상세) |
| 이 모임에서의 내 역할 상세 | 역할에 대한 깊이 있는 설명 |
| 인싸 전략 | "이 모임에서 인싸가 되려면?" 맞춤 조언 |
| 주의할 관계 | "이 사람과는 이런 점을 조심하세요" |
| 나만의 강점 | 이 그룹에서 내가 빛나는 포인트 |

> **프라이버시 원칙:** 개인 결과는 절대 다른 참여자에게 노출되지 않는다. "주의할 관계" 같은 민감한 내용은 개인 화면에서만 제공.

### 3-8. 결과 저장 및 공유

**저장:**
- 결과는 영구 저장되어 "궁합 탭 > 내 궁합 기록"에서 언제든 다시 확인 가능
- 향후 관계맵 기능 출시 시 자동으로 반영 (시드 데이터)

**공유:**
- [링크 공유] 버튼 → 앱 딥링크 생성
- 미가입자가 링크 클릭 시 → 앱 설치 유도 → 가입 후 결과 확인 가능 (바이럴 루프)
- 공유 시 공유 결과만 노출 (개인 결과는 공유 불가)

### 3-9. 화면 흐름 (Screen Flow)

```
[하단 탭: 궁합]
    │
    ├── 궁합 메인
    │   ├── [1:1 궁합] → 기존/별도 화면
    │   ├── [그룹 궁합 만들기] CTA
    │   └── 내 궁합 기록 목록
    │
    ├── 방 생성
    │   ├── 모임 카테고리 선택
    │   ├── 방 이름 입력
    │   ├── 인원 설정 (3~5)
    │   └── [방 만들기]
    │
    ├── 대기실
    │   ├── 초대코드 / QR / 링크
    │   ├── 참여자 실시간 목록
    │   └── 방장 관리 (인원변경/내보내기/삭제)
    │
    ├── 결제
    │   ├── 결제 방식 선택
    │   ├── HANI 재화 결제
    │   └── 결제 완료 대기
    │
    └── 결과
        ├── 공유 결과 탭
        │   ├── 그룹 케미 점수
        │   ├── 관계 매트릭스
        │   ├── 역할 카드
        │   └── [링크 공유]
        └── 내 결과 탭
            ├── 개인 상세 궁합
            ├── 인싸 전략
            └── 주의 관계 / 강점
```

## 5. 수익 모델 연계

### 사주일기 → HANI PASS 전환 유도

```
[무료]                          [HANI PASS]
 기본 기록 기능                    ✅ 기본 기록 기능
 감정 그래프                       ✅ 감정 그래프
 캘린더 히트맵                     ✅ 캘린더 히트맵
 십성별 분포                       ✅ 십성별 분포
 ─────────────────               ✅ AI 패턴 분석 인사이트
                                  ✅ AI 챗봇이 내 일기 참조하여 상담
                                  ✅ 미래 예측 (과거 패턴 기반)
                                  ✅ 상세 리포트 자동 생성
```

**전환 포인트:**
- 30일 기록 달성 시: "30일 데이터가 모였어요! AI가 당신의 패턴을 발견했습니다" → HANI PASS 유도
- 과거 사건 5개 이상 기록 시: "과거 사건을 분석하면 미래를 예측할 수 있어요" → HANI PASS 유도
- AI 챗봇 대화 중: "사주일기 데이터와 연동하면 더 정확한 분석이 가능해요" → 연동 유도

### 다자간 궁합 → 직접 수익

- 인당 HANI 재화 소비
- 모임 후 개별 1:1 궁합으로 전환 유도 (추가 결제)
- 결과 공유 링크를 통한 바이럴 → 신규 유저 확보 → HANI PASS 전환

## 6. 향후 확장 계획

### 6-1. 관계맵 (트리/버블 형태)

다자간 궁합의 결과 데이터를 시드로 활용하여, 사용자를 중심으로 한 인간관계 지도를 시각화한다.

- 버블 크기: 궁합 점수 / 관계 깊이
- 연결선 색상: 관계 유형 (친구/연인/동료 등)
- 클러스터링: 모임 단위로 그룹핑
- 궁합 점수 변화 추이 (시간에 따라)

### 6-2. 사주일기 확장

- 일운 정확도 검증 데이터 축적 → 명리학 연구 기여 가능성
- 사용자 간 익명 통계: "같은 상호작용을 겪은 다른 사용자들은 이런 경험을 했어요"
- 시운(時運) 추가: 시간 단위 운세 (고급 사용자용)

### 6-3. 다자간 궁합 확장

- 인원 확대: 최대 10인 이상
- 팀 빌딩 리포트: 직장/사업 카테고리에서 팀 구성 최적화 제안
- 정기 모임 트래킹: 같은 그룹이 반복 궁합 시 변화 추이 제공
